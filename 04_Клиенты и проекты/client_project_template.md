# Карточка клиента и проекта

## Назначение файла

Карточка клиента фиксирует всю рабочую информацию по клиенту и проекту: задачу, предложение, этапы, оплату, материалы, риски, договоренности и историю контактов.

## Когда использовать

- Лид стал заинтересованным.
- Обсуждается конкретная задача.
- Клиент просит цену или КП.
- Согласован первый этап.
- Проект переходит в работу.

## Как использовать

Скопировать шаблон в отдельный файл клиента, переименовать по правилам названий и заполнять по мере движения проекта. После каждого важного контакта обновлять статус и следующий шаг.
