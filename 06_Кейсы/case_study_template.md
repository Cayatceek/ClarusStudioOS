# Шаблон кейса

## 1. Назначение кейса

Кейс показывает, как студия разобрала задачу клиента, предложила решение и довела работу до понятного результата. Кейс нужен для портфолио, контента, доверия и будущих продаж.

## Когда использовать

- После завершения проекта.
- После завершения важного этапа.
- Когда клиент дал отзыв.
- Когда есть разрешение на публикацию.
- Когда проект можно использовать как пример для похожих клиентов.

## Как использовать

Заполнить полный шаблон, затем подготовить короткие версии для Instagram, Telegram, сайта и портфолио. Перед публикацией проверить разрешение клиента и убрать данные, которые нельзя показывать.

## 2. Когда оформлять кейс

Кейс оформляется, когда есть:

- понятная исходная ситуация;
- конкретная проблема;
- задача;
- решение;
- список сделанных работ;
- результат;
- выводы или следующий этап;
- согласие клиента на публикацию.

Не нужно ждать идеального проекта. Небольшой мини-сайт, форма заявки или каталог тоже может быть хорошим кейсом, если он показывает рабочий подход.
