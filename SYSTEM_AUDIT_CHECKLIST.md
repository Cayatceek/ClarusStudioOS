# Чеклист аудита системы

## Назначение файла

Этот чеклист используется раз в месяц, чтобы проверить состояние Clarus Studio OS: CRM, шаблоны, КП, проекты, кейсы, финансы, архив, контент, названия файлов и документы.

## Когда использовать

- В конце месяца.
- После нескольких активных проектов.
- Если в системе снова появился хаос.
- Перед обновлением шаблонов и регламентов.

## Как использовать

Пройти по разделам, отметить пункты, записать проблемы и выбрать 3-5 улучшений на следующий месяц.
